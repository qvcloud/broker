# 安全策略 (Security Policy)

## 支持的版本

目前仅对最新主版本提供安全更新。

| 版本 | 支持状态 |
| :--- | :--- |
| v1.x | ✅ 支持中 |
| < v1.x | ❌ 不再支持 |

## 报告漏洞

如果你在本项目中发现了安全漏洞，请**不要**通过公开的 Issue 报告。请通过以下方式联系：

*   发送邮件至: [security@example.com](mailto:security@example.com) (请替换为实际地址)
*   或是通过 GitHub 的 [Private Vulnerability Reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-security-advisories/privately-reporting-a-security-vulnerability) 功能。

我们会在 48 小时内确认收到报告，并在此后提供初步回复及修复计划。
